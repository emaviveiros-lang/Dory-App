# Dory-App
Dory App 🐟
A personal task, issue and note tracker that lives in your browser — just keep swimming through your to-dos.
Features

Add tasks, issues and notes in one place
Set custom reminder intervals per item (every X minutes, hours or days)
Desktop notifications with sound alerts until you mark things done
Edit items and add status notes to track progress
Filter by type — tasks, issues, notes or done
Works from any device via your personal GitHub Pages link

How to use

Open your Dory App link in Chrome
Click "enable reminders" and allow notifications
Add your first task and set a reminder interval
Pin the tab so it's always one click away

Tech
Built as a single HTML file with vanilla JavaScript. No installs, no accounts, no servers. Your data is saved locally in your browser.
